# git-demo

An empty starter project for demonstrating Git capabilities — branching, commits, pull requests, and more.

## Getting started

```bash
git clone https://github.com/rosseca/git-demo.git
cd git-demo
```

## What this is for

A sandbox to practice and demonstrate:

- Creating commits and writing good commit messages
- Branching and merging
- Pull requests and code review
- Tags and releases

## Example: greeting script

A small Python script demonstrates the workflow:

```bash
python3 greet.py            # -> Hello, World!
python3 greet.py Vlad       # -> Hello, Vlad!
```

Run the tests:

```bash
python3 -m pytest           # or: python3 -c "import test_greet; ..."
```

Happy hacking! 🚀
