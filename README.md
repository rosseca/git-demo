# git-demo

An empty starter project for demonstrating Git capabilities — branching, commits, pull requests, and more.

## Getting started

```bash
git clone https://github.com/rosseca/git-demo.git
cd git-demo
```

## What this is for

A sandbox to practice and demonstrate:

- Creating commits and writing good commit messages
- Branching and merging
- Pull requests and code review
- Tags and releases

Happy hacking! 🚀
